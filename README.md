## 🚀 Featured Project

### 🚌 [Naija Transport — Route & Fare Finder](https://naijatransport.vercel.app)

> A full-stack, real-time Nigerian transport platform — my most ambitious project to date.

- 🔐 JWT authentication with refresh tokens + httpOnly cookies
- 🗺️ Geospatial route & stop search with MongoDB 2dsphere indexes
- ⚡ Real-time incident map updates via Socket.IO
- 💰 Crowdsourced fare reporting engine
- 📍 GPS trip recording & replay
- 🤖 WhatsApp bot integration for report ingestion
- 🏆 Gamification — points, streaks, badges, leaderboard
- 🔄 Full CI/CD with GitHub Actions → Vercel + Render
